# СИСТЕМА_MVP

Технический каркас MVP производственного контура учетной системы.

## Назначение

Эта папка содержит будущую веб-систему:

- единое ядро данных;
- роли и доступы;
- справочники;
- смены;
- назначения техники;
- рейсы;
- простои;
- отчеты.

## Текущий статус

Создан первый Django-каркас проекта и базовые модули:

- `core` - ядро и служебные функции;
- `users` - сотрудники, роли и доступы;
- `references` - справочники;
- `shifts` - смены и вахты;
- `assignments` - назначения техники;
- `trips` - рейсы;
- `downtimes` - простои;
- `reports` - отчеты и шаблоны.

## Локальный запуск

```powershell
.\.venv\Scripts\python.exe backend\manage.py runserver
```

Перед первым запуском нужно выполнить миграции:

```powershell
.\.venv\Scripts\python.exe backend\manage.py migrate
```
