# Быстрый UI-режим Codex

Дата: 08.07.2026

Создан документ `FAST_UI_MODE.md` для задач типа A — мелкая UI-правка. Режим ограничивает работу Codex одним рабочим контуром, запрещает анализ всего проекта без необходимости и фиксирует минимальные проверки для template/CSS/view-правок.

Обновлены инструкции:

- `AGENTS.md` — добавлено правило использовать `FAST_UI_MODE.md` при запросах "быстро", "мелкая правка", "поменять кнопку", "поправить интерфейс", "переставить блок";
- `CODEX_WORKFLOW.md` — добавлен раздел "Быстрая UI-правка за один короткий цикл";
- `СИСТЕМА_MVP/backend/AGENTS.md` — уточнено, что для типа A сначала ищутся конкретные `template`/`static`, а `view` трогается только если без этого невозможно.

Код, UI, templates, views, static, models, migrations, база данных, `.env`, production и deploy не менялись. Django-проверки, локальный сервер, `migrate`, `collectstatic`, `push` и deploy не запускались.
