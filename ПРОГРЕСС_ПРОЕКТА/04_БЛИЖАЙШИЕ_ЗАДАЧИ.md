# Ближайшие задачи

## Главная ближайшая цель

Довести рабочие интерфейсы MVP до пилотного состояния и подготовить первый демонстрационный сценарий.

## Текущая рабочая сессия

Сбор ответов на 20 вопросов по производственному контуру завершен. Ответы зафиксированы в файле `05_ВОПРОСЫ_ПРОИЗВОДСТВЕННОГО_КОНТУРА.md`.

Следующий шаг: собрать на основе этих ответов документ `ТЗ_MVP_ПРОИЗВОДСТВЕННОГО_КОНТУРА.md`.

## Ближайший порядок работы

1. Исправить найденные неудобства демонстрационного цикла.
2. Развивать конструктор отчетов: добавить более специальные формы Excel под заказчика и позже расширить набор расчетных показателей.
3. Улучшить витрину руководства: добавить более наглядные графики, динамику по датам и сравнение день/ночь.
4. Проверить, какие данные нужны диспетчеру для ежедневного использования, а какие можно оставить на следующий этап.
5. Позже отдельно проработать интерфейсы каждого сотрудника: водитель, машинист экскаватора, горный мастер, диспетчер, механик, администратор, руководство.
