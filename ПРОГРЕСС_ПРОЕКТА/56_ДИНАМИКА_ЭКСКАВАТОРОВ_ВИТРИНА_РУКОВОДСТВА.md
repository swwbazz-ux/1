# Динамика экскаваторов в витрине руководства

Дата: 2026-07-07.

## Что зафиксировано

- Экран `Руководство MVP -> Динамика` остается частью витрины руководства и не должен расходиться со стилем соседних вкладок.
- Переключатель режима графика размещается внутри шапки графика, чтобы не перегружать верхнюю панель фильтров.
- Поддерживаемые режимы графика: `Накопительно м3`, `Темп м3/час`, `Рейсы`.
- Основные KPI, таблица и сводные строки отчета считаются по прежним правилам отчета.
- События `truck_loaded` используются для более точной событийной линии графика, но не должны переносить рейс между отчетными бакетами.
- Для часового графика одна смена отображает полную шкалу: дневная `07:00 -> 19:00`, ночная `19:00 -> 07:00`.
- Для длинных часовых периодов подписи оси X разрежаются, чтобы график не становился нечитаемым.

## Проверено

- `manage.py check`.
- Целевые тесты отчета динамики экскаваторов.
- Браузерная проверка desktop/mobile на локальном сервере: переключатель режимов, 4 KPI, SVG-график, заливка под графиком, отсутствие горизонтального overflow.
