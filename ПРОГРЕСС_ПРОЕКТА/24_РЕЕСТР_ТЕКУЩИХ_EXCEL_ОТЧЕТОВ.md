# Реестр текущих Excel-отчетов для сверки MVP

Дата создания: 19.06.2026

## Назначение

Документ фиксирует найденные Excel-файлы из текущей работы диспетчерской и показывает, какие из них уже частично или полностью покрываются MVP.

Цель: перед пилотным запуском понимать не просто список экранов системы, а связь между старой Excel-отчетностью и новой веб-системой.

Текущий статус по основной дорожной карте: этап 9 из 10, готовность примерно 99%.

## Поля, которые уже обнаружены в файле анализа

В файле `Анализ_отчетов_упорядоченный (1).xlsx` уже собрана матрица видов данных. Основные группы:

- техника: самосвал, экскаватор, гаражный номер, модель, VIN/идентификационный номер;
- смена и дата;
- рейсы: количество рейсов, часовой интервал, объем за час, объем за смену, плечо, пункт разгрузки;
- экскаваторы: горизонт, блок, плановое задание, выполненный объем, простои, перегоны, смена фронта работ;
- простои: вид простоя, категория простоя, длительность, комментарий, признак простоя с включенным ДВС;
- ремонты и ТО: вид ТО, ЕТО, категория ремонта, вид неисправности, время ремонта;
- показатели техники: ФВР, время в работе, КИО, КТГ, пробег, моточасы, топливо;
- отчетность заказчику: объемы, смены, пункты разгрузки, отклонения, примечания.

## Предварительный вывод

MVP уже закрывает основу отчетного контура:

- рейсы;
- объемы;
- смены;
- суточный отчет;
- Excel-выгрузки;
- механические простои;
- витрина руководства.

Но для полного замещения Excel-диспетчерской после пилота нужно будет отдельно доработать:

- точную старую почасовую матрицу, если диспетчерской не хватит группировки по часу;
- КИП/КТГ и КИО/КТГ;
- расчет невыполненного объема по простоям ККД/СКДР, если он потребуется после пилота;
- перегоны экскаваторов;
- график ТО.
- отдельная модель плотностей по карьеру, если заказчик потребует считать тоннаж с учетом карьера.
