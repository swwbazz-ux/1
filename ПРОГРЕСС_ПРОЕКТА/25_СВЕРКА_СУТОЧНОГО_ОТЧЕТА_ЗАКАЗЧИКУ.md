## Структура старой суточной формы

На дневном листе старая форма содержит:

- заголовок отчета о работе ООО "Коппер Рисорсез" на ООО "Амур Минералс";
- отдельную таблицу I смены, дневной 08:00-20:00;
- отдельную таблицу II смены, ночной 20:00-08:00;
- блок `Работа выемочного оборудования`;
- поля: тип грунта, наименование/номер экскаватора, плановое задание, объем выполненных работ, горизонт погрузки, блок, место разгрузки, плечо транспортировки, простои, примечание;
- суточную сводку с планом, фактом, отклонением, итогами за сутки и с начала месяца;
- итоги по видам материалов: горная масса, добыча руды, сульфидная, переходная, вскрыша, окисленная, рыхлая, скальная, ПСП, ППСП;
- средневзвешенное плечо транспортировки.

В странице `/reports/customer-daily/` и Excel-выгрузке `/reports/customer-daily/export/` уже есть:

- заголовок суточного отчета;

- строки по породе, экскаватору, плану, факту, горизонту, блоку, месту разгрузки, плечу, простоям и примечанию;
- итоги по породам;

## Основные расхождения

1. Средневзвешенное плечо пока не вынесено в отдельную сводку.
2. Расчетная вкладка по самосвалам с полями `рейсы`, `км`, `объем`, `итог`, `м3*км` пока покрыта только частично через отчет по объемам и конструктор отчетов.
3. Производственные простои экскаватора пока не имеют полноценного отдельного модуля статусов экскаватора.
4. Справочник пород нужно сверить с названиями заказчика, чтобы группировки в отчете совпадали с привычной формой.
5. Месячный план пока считается по плановым заданиям в рейсах, а не через отдельную модель месячного плана.

## Решение для первого пилота

Для первого пилота текущий суточный отчет можно использовать как основу, потому что он уже закрывает главный рабочий смысл:

- сменные объемы;
- суточные итоги;
- план/факт/отклонение;
- разрез по породам;
- разрез по экскаваторам и точкам разгрузки;
- выгрузку в Excel.

Перед промышленным запуском нужно отдельно доработать:

- средневзвешенное плечо;
- точный расчет `м3*км`;
- отдельный шаблон расчетной вкладки по самосвалам;
- статусы и производственные простои экскаваторов;
- справочник пород в названиях заказчика.
